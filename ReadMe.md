# メトロ用乗降促進プラグイン
このプラグインは乗降促進を再現するプラグイン。メトロ総合プラグインとの併用が前提。

## SWインデックス
ATS_D（2キー）…乗降促進鳴動（長押し鳴動、離すと放送）

## その他
マスコンキーの入力がない場合メトロのものが鳴ります。汎用乗降促進として使う場合は33,34のみを使用してください。
